LAB2-Shell

Это оболочка терминала в Linux
Как использовать: Запуск программы

bash
python -m venv .venv

source .venv/bin/activate  # Linux/macOS
# или
.venv\Scripts\activate     # Windows
Запуск терминала

bash
python src/main.py

Запуск тестов

bash
pytest tests/

Допущения:
Сделано под Linux
